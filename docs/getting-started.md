## Installation

Neko can be installed on Windows, macOS, and Linux.

### Download

Visit the [Releases page](https://github.com/GitNekoTeam/neko/releases) to download the latest version for your platform:

- **Windows**: `.exe` installer (System or User) or `.zip` portable
- **macOS**: `.zip` (x64, ARM64, or Universal)
- **Linux**: `.deb`, `.rpm`, `.tar.gz`, or `.AppImage`

### System Requirements

- Windows 10/11 (x64, ARM64)
- macOS 10.15+ (x64, ARM64)
- Linux (x64, ARM64) with glibc 2.28+

## First Steps

After installing Neko, here are some first steps to get started:

1. **Open a folder**: Use File > Open Folder to open your project
2. **Install extensions**: Click on the Extensions icon in the sidebar to browse and install extensions
3. **Configure settings**: Use File > Preferences > Settings to customize your editor
4. **Try AI features**: Use the Neko AI sidebar to interact with the AI assistant

## Basic Usage

Neko is based on VS Code, with additional AI capabilities:

- Full VS Code compatibility
- Built-in AI assistant for code generation and explanation
- Open VSX extension marketplace
- No telemetry by default

## AI Features

Neko includes a built-in AI assistant that can help you:

- Generate code from natural language descriptions
- Explain existing code
- Refactor and improve code quality
- Answer programming questions
- Debug issues

Access the AI assistant through the Neko AI sidebar or use keyboard shortcuts.

## Keyboard Shortcuts

Here are some essential keyboard shortcuts to get you started:

| Action | Windows/Linux | macOS |
|--------|---------------|-------|
| Quick Open | `Ctrl+P` | `Cmd+P` |
| Command Palette | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| Settings | `Ctrl+,` | `Cmd+,` |
| Keyboard Shortcuts | `Ctrl+K Ctrl+S` | `Cmd+K Cmd+S` |
