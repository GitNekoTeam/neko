# Welcome to Neko AI

Neko is an AI-powered code assistant that helps you write, understand, and improve your code.

## Getting Started

1. Open the Neko panel from the sidebar
2. Configure your API provider
3. Start chatting with the AI

## Key Features

- Multi-provider support (OpenAI, Anthropic, etc.)
- Code generation and editing
- Context-aware assistance
- File and folder operations

Click "Next" to learn how to configure your API provider.
