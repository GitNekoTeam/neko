# Tools and Commands

Neko AI has access to various tools to help you.

## Available Tools

- **Read File** - Read file contents
- **Write File** - Create or modify files
- **List Files** - Browse directory structure
- **Search Files** - Find files by pattern
- **Execute Command** - Run terminal commands
- **Browser Action** - Interact with web pages

## Slash Commands

Type `/` in the chat to see available commands:

- `/clear` - Clear chat history
- `/new` - Start a new task
- `/mode` - Change agent mode
- `/help` - Show help

## Tips

- Be specific in your requests
- Use context mentions for better results
- Review changes before approving
